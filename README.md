## [DEMO-Click Me](https://dark-light-theme-context-sertacgltkn.netlify.app)

### Getting started

1. Clone this repo
2. Run `npm install`
3. Run `npm start`
4. Visit [http://localhost:3000](http://localhost:3000) to see the app running
